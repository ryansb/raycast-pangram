# raycast-pangram

This is an ironically AI-generated extension for [raycast](https://raycast.com). It sends selected text or clipboard contents to Pangram and returns an AI detection result.

Detection results are retained for 7 days and markdown-formatted.

API keys require paid usage from [Pangram](https://www.pangram.com/).
